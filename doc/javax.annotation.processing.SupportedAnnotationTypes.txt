*javax.annotation.processing.SupportedAnnotationTypes_Description*

An annotation used to indicate what annotation types an annotation processor supports. The 
(|javax.annotation.processing.Processor|) method can construct its result from the value of this annotation, as done by 
(|javax.annotation.processing.AbstractProcessor|) . Only strings conforming to the 
grammar(|javax.annotation.processing.Processor|) should be used as values.
