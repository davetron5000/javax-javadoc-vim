*javax.lang.model.element.UnknownElementException_Description*

Indicates that an unknown kind of element was encountered. This can occur if the language evolves and new kinds of elements are 
added to theElementhierarchy. May be thrown by an element visitor(|javax.lang.model.element.ElementVisitor|) to indicate that 
the visitor was created for a prior version of the language.
