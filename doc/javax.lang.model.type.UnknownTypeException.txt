*javax.lang.model.type.UnknownTypeException_Description*

Indicates that an unknown kind of type was encountered. This can occur if the language evolves and new kinds of types are added 
to theTypeMirrorhierarchy. May be thrown by a type visitor(|javax.lang.model.type.TypeVisitor|) to indicate that the visitor 
was created for a prior version of the language.
