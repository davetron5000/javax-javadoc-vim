*javax.management.Attribute_Description*

Represents an MBean attribute by associating its name with its value. The MBean server and other objects use this class to get 
and set attributes values. 



*javax.management.Attribute(String,Object)*

public Attribute(
  java.lang.String name,
  java.lang.Object value)

Constructs an Attribute object which associates the given attribute name with the given value. 

    name - A String containing the name of the attribute to be created. Cannot be null. 
    value - The Object which is assigned to the attribute. This object must be of the same type as the attribute.
