*javax.script.SimpleScriptContext_Description*

Simple implementation of ScriptContext. 



*javax.script.Bindings_javax.script.SimpleScriptContext.engineScope*

This is the engine scope bindings. By default, a SimpleBindings is used. Accessor methods setBindings, getBindings are used to 
manage this field. 


*java.io.Writer_javax.script.SimpleScriptContext.errorWriter*

This is the writer to be used to output errors from scripts. By default, a PrintWriter based on System.err is used. Accessor 
methods getErrorWriter, setErrorWriter are used to manage this field. 


*javax.script.Bindings_javax.script.SimpleScriptContext.globalScope*

This is the global scope bindings. By default, a null value (which means no global scope) is used. Accessor methods 
setBindings, getBindings are used to manage this field. 


*java.io.Reader_javax.script.SimpleScriptContext.reader*

This is the reader to be used for input from scripts. By default, a InputStreamReader based on System.in is used and default 
charset is used by this reader. Accessor methods getReader, setReader are used to manage this field. 


*java.io.Writer_javax.script.SimpleScriptContext.writer*

This is the writer to be used to output from scripts. By default, a PrintWriter based on System.out is used. Accessor methods 
getWriter, setWriter are used to manage this field.

public |java.lang.Object| getAttribute(java.lang.String name)

Retrieves the value of the attribute with the given name in the scope occurring earliest in the search order. The order is 
determined by the numeric value of the scope parameter (lowest scope values first.) 


    name - The name of the the attribute to retrieve. 

    Returns: The value of the attribute in the lowest scope for which an attribute with the given name is defined. Returns null if no 
             attribute with the name exists in any scope.
