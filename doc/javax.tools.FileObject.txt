*javax.tools.FileObject_Description*

File abstraction for tools. In this context, file means an abstraction of regular files and other sources of data. For example, 
a file object can be used to represent regular files, memory cache, or data in databases. 

All methods in this interface might throw a SecurityException if a security exception occurs. 

Unless explicitly allowed, all methods in this interface might throw a NullPointerException if given anullargument.
