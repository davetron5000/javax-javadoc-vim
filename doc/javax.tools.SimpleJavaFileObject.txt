*javax.tools.SimpleJavaFileObject_Description*

Provides simple implementations for most methods in JavaFileObject. This class is designed to be subclassed and used as a basis 
for JavaFileObject implementations. Subclasses can override the implementation and specification of any method of this class as 
long as the general contract of JavaFileObject is obeyed.
